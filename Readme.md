# Учебный проект UBER

## Файлы и папки:

* [.editorconfig](https://editorconfig.org/) - Файл с настройками для редактора. VS Code предложит установить расширение.
* .vscode/extensions.json - Порекомендует установить editorconfig для VS Code.
* [.gitignore](https://git-scm.com/docs/gitignore) - Список файлов и расширений, которые Git должен игнорировать.
* css/common и css/blocks - Файлы с общими стилями и стилями для блоков. Подключаются с помощью @import в style.css.
* script/js - Выведет приветственное сообщение в консоль.
